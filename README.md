# css-only-40-ways-hello-world
JavaScriptを使わず、HTMLとCSSだけのあらゆる手段を使って単一の`index.html`ファイルだけで40通りの方法で`hello world`を
```
hello
world
```
のように2行表示で表示させる。

面倒すぎる方法（`clip-path`で全部切り抜く、`transform`や`border-radius`、`corner-shape`で全ての線を作るなど）や、`svg`要素関連、`math`要素関連は除外した。
